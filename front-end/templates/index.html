Important things to note

- A week of Hashtags with #Balr time frame is 7 days. This time frame was chosen due to limits of request through twitter API and limiting server run time (Free tier AWS)

- Location data for Twitter is user generated and not an indication of actual loction, tweet language is used as a proxy for location.

- Total potential reach, is the sum of followers for unique accounts using the #Balr hashtag this week. If a user posts twice in 1 week only one set of followers will be counted.

- All data for this dashboard is publically available through Twitter API

- Source code this project is available at Github
